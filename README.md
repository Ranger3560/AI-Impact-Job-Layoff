# AI-Impact-Job-Layoff

AI Workforce Layoff Risk Analysis

This project explores how AI adoption, task automation, employee skills, and workplace characteristics influence layoff risk. Using data analysis, visualization, and machine learning techniques, the study identifies key factors associated with workforce displacement and builds predictive models to classify employees into Low, Medium, or High layoff risk categories.

Key Features:

Exploratory Data Analysis (EDA)
AI adoption and automation trend analysis
Layoff risk prediction using machine learning
Feature importance analysis
Data visualizations and insights

Technologies: Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

Goal: Provide data-driven insights into the impact of AI on workforce stability and help organizations make informed upskilling and workforce planning decisions.
